# Employee-Management-System
 Develop a single page application(SPA) using Angular 8 as a front-end and Spring boot restful API as a backend.
 
 
# Features Implementation:

          Create an Employee
           Update an Employee
           List of Employees
           Delete Employee
          View Employee
